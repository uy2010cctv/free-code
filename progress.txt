## Codebase Patterns
- SessionManager stores session workspace paths using `resolve(this.cwd, 'workspace', id)` format
- The workspace folder is created via `mkdir(resolve(this.cwd, 'workspace', id), { recursive: true })`
- Document tools (WebWordTool, WebExcelTool) should use workspace-relative paths for document operations
