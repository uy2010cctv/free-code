## Codebase Patterns
- SessionManager stores session workspace paths using `resolve(this.cwd, 'workspace', id)` format
- The workspace folder is created via `mkdir(resolve(this.cwd, 'workspace', id), { recursive: true })`
- Document tools (WebWordTool, WebExcelTool) should use workspace-relative paths for document operations
- DocumentManager class provides unified document APIs: createDocument, listDocuments, readDocument, writeDocument
- Document file extensions: .docx/.doc for Word, .xlsx/.xls for Excel
- docx library uses TextRun for text formatting (bold, italic, underline, color, size)
- Paragraph alignment via AlignmentType enum (CENTER, LEFT, RIGHT, JUSTIFIED)
- Tables created with Table, TableRow, TableCell classes from docx
- xlsx library uses XLSX.WorkBook, XLSX.WorkSheet for Excel files
- xlsx cell formulas set via cell.f property, styles via cell.s property
- xlsx column/row dimensions set via worksheet['!cols'] and worksheet['!rows']
- xlsx cell merges set via worksheet['!merges'] array
- docx-preview renders Word documents in browser via renderAsync function
- Preview components should detect file type from extension and render appropriately
- Fullscreen API uses requestFullscreen() and fullscreenchange event
- Browser download via Blob + URL.createObjectURL + anchor click
